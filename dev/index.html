AI應用說明

以下評比基於本次 Hackathon II 專案的使用體驗，除了 Claude 為 Pro Plan 外，其他均為免費版本。

本次 Hackathon II 專案，我採用「多 AI 協作」的模式，將各平台視為我的 Team Member，讓它們之間互相除錯並提供解決方案，而我則扮演 Multi-Context Processing (MCP) 的角色，負責串聯、統整及修正討論結果。

以下為個人實際使用的心得評價。

AI平台比較

能力面向🥇 第一名🥈 第二名🥉 第三名未列入評比/備註
對話親和度 ChatGPT Gemini Claude Pro Grok、Deepseek
程式碼能力 Gemini ChatGPT Claude Pro Grok、Deepseek
文案與寫作 Claude Pro Gemini ChatGPT
圖像生成 Gemini Grok Gemini 有浮水印，Grok 無浮水印
影片生成 Grok
3D 物件生成 Tripo3D MeshyAI Hitem3D
Gemini：全面能力大幅提升，專案貢獻度最大
  • 🟢 優勢：在本次專案中提供了最大幫助，程式編纂能力領先 ChatGPT 與 Claude。
  • 🔴 劣勢：程式操作與偵錯描述常過於專業或簡略，可能使不熟悉語言的使用者困惑。
  • ⚪ 潛力：若影片生成能力成熟，將成為下一次付費訂閱首選。
ChatGPT：最容易溝通與理解的 AI 工具
  • 🟢 優勢：語氣親和、對話壓力最低，互動體驗最佳。
  • 🔴 劣勢：程式優勢漸弱，文案易略顯敷衍。
Claude（Pro Plan）：文案最強，但工程能力不足
  • 🟢 優勢：文案寫作細膩、品質極好。
  • 🔴 劣勢：
    • Token 設計差：會把已生成的答案收回，導致需重複提問、浪費 Token。
    • MCP 表現未達預期：精準度有限且消耗大量 Token。
    • 程式碼錯誤率偏高，工程水準未達專業等級。
Grok：免費影片生成的最佳選擇
  • ⚪ 定位：主要作為圖片與影片生成工具。
  • 🟢 優勢：免費額度大方、影片品質不錯且無浮水印。
Deepseek：鬼打牆時的奇兵
  • ⚪ 定位：用於打破 Gemini／ChatGPT／Claude 討論僵局。偶爾提供完全不同的解法，有時非常有效。
3D 物件生成工具比較
  • Tripo3D：生成物件精緻度最高。
  • MeshyAI：免費額度無法下載 3D 模型。
  • Hitem3D：生成物件較為粗糙。
