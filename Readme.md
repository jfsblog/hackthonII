本作品為一款以街貓為主題的小型互動遊戲，提供簡單的場景探索與任務流程。

> 在城市的角落裡，有一群不曾被忽視、卻常被遺忘的小生命。
> 牠們穿梭在你的世界邊緣，靜靜等待一束願意停下的光。

* 追尋迷途的小貓
* 透過探索與對話了解牠們
* 完成互動任務
* 傳達包容、陪伴、領養代替購買的寓意

## 操作方式

遊戲採用基本的 3D 探索操作：

| 動作         | 控制                |
| ---------- | ----------------- |
| 移動         | **W / A / S / D** |
| 跳躍         | **Space**         |
| 互動／對話／點擊物件 | **滑鼠左鍵**          |
| 視角移動       | **滑鼠移動**          |
